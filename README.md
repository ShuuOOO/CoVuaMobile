# ♟️ CoVuaMobile – Chess Mobile Application

Android chess application developed using **Java** and **Android Studio**, implementing standard chess rules with a clean and intuitive mobile interface.  
This project focuses on **game logic implementation**, **object-oriented design**, and **mobile application deployment**.

---

## Features

- Basic chess gameplay for two players on the same mobile device
- Core chess logic implementation including legal move validation, piece capturing, and basic check/checkmate detection
- Simple and intuitive user interface optimized for mobile screens
- Clear separation between game logic and UI code for easier readability and maintenance 

---

## Tech Stack

- **Language:** Java  
- **Platform:** Android  
- **IDE:** Android Studio  
- **Build Tool:** Gradle  
- **Minimum SDK:** API 21 (Android 5.0)  
- **Target SDK:** API 35  

---

### Requirements

Before running the project, make sure your environment meets the following requirements:

- Android Studio (latest stable recommended)
- Java Development Kit (JDK) 11
- Android SDK  
  - Minimum SDK: API 21  
  - Target SDK: API 35
- Android Emulator **or** Physical Android Device
