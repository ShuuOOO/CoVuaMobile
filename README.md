# CoVuaMobile
This is a native Android application developed using Java.

Primary Language: Java (Java 11)

UI: XML (Android layout and resources)

Build System: Gradle

Platform: Android SDK (minSdk 21, targetSdk 35)

The application follows a traditional Android native architecture, with clear separation between activities, controllers, and models.
All game logic and UI handling are implemented without using cross-platform frameworks or game engines.
